# Project Overview


The repository contains my Digital Literacy Portfolio project undertaken as part of the Digital Literacy course. The project showcases my understanding of various digital tools, professional online presence, safe communication practices, collaboration tools, and measures to prevent cybercrime. Each task has been stored in separate folders with relevant files and images.

Task 1. Digital Literacy Infographic


Developed an infographic with one page of information on digital literacy, useful tools, internet safety, professional online presence, and email usage. The infographic was created using Canva and added to the repository accordingly.

Task 2. Student Digital Portfolio


Developed professional profiles on GitHub, LinkedIn, and Kaggle. These websites enable students to have an online presence and track their progress.

Task 3. Platforms Exploration


Completed a beginner coding challenge on HackerRank. Created a Google Form quiz on Digital Literacy Awareness and linked the form to Google Sheets.

Task 4. Email Etiquette
Developed two professional email drafts. Created a social media do and don't list.

Task 5. Cybercrime Awareness
Developed a case study on cybercrime and a checklist on its prevention, focusing on online safety.
